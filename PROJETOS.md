## Detalhamento: dev_task_manager

**Stack:** .NET 9, ASP.NET Core WebAPI (C#), React 19, SQLite. Bring your own API key (Ollama / OpenAI / Gemini).

**O que faz:**

- **Administrador de tarefas** com boards Kanban e notificações integradas do Windows.
- **Subtarefas** criadas e enriquecidas com **IA agentica**: descrições dos cards, sugestões de próximos passos.
- **Insights do dia**: o que priorizar e como agir no dia.
- **Insights por card**: por onde priorizar, o que fazer e como executar em cada card.
- Foco em produtividade e priorização guiada por IA, mantendo a chave de API sob controle do usuário (local / bring your own key).

**Links:**

- Demo (React no GitHub Pages): https://wmakeouthill.github.io/dev_task_manager/
- Repositório: https://github.com/wmakeouthill/dev_task_manager
